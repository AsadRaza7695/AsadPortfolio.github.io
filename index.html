Face Recognition Attendance System

Our project is a real-time face recognition system using the "face_recognition" library and OpenCV. It starts by loading a set of known face images from a specified directory and storing them along with their corresponding class names. The code then captures video frames from a webcam, detects faces in the frames, and compares them with the known faces using face encodings. When a match is found, it marks the attendance of the recognized individual by appending their name and timestamp to a CSV file. To accomplish this, the project utilizes the "face_recognition" library, which simplifies face detection, facial feature extraction, and face recognition tasks. The library employs a pre-trained deep learning model to detect faces, identify facial landmarks, and extract face encodings. It then compares the computed face encodings with the known encodings to determine if there is a match. Project provides a convenient way to perform real-time face recognition by leveraging the "face_recognition" library.
